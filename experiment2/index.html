Experiment 2 - Living Impression

Description

This project offers a generative, living impression of this inspiring image:

  • Source

    • Technical

    - Color: The sky, trees, and mountains colors were sampled from the inspiring image using the [Image Color Picker tool](https://imagecolorpicker.com/en).

    - Primitives: The trees are rendered with `triangle` primitives.

    - Polygons: The background mountain is a single randomly polygon using `vertex` calls.

    - Composition: Different random seed values lead to differently shaped mountains and differently positioned trees.

    - Life: The position of trees is animated over time as if to simulate the view from a passing train and the slightly responds to the horizontal motion of the mouse cursor.

    - ArtStyle: all shapes were drawn using a painted style where one layer is added on top of another to shwo depth in the image.

    Reflection

    The biggest obstacle I encountered was the formating of the index.html and the setup() of project.js. After this experiement I will never use raw numbers in my draw() functions because it will look completely diffrent if i have a diffrent canvas size than the one i was experimenting on.
